CN Visual



O CN Visual foi idealizado na forma de um pacote a ser integrado ao software R, um programa gratuito, de código aberto e altamente extensível.


Instruções de instalação:
  1. Baixe e instale o programa R
  2. Baixe pacote: CNvisual. Não é necessário descompactar o arquivo.
  3. Instalação do CNvisual:
    • Abra programa R e, no menu, siga os seguintas passos:
      • Ferramentas -> Instalar pacotes -> Selecioce a opção: Instalar a partir do Repositório (CRAN) -> Na caixa de texto digite: RGtk2, gWidgetsRGtk2.
      • Ferramentas -> Inatalar pacotes -> Selecioce a opção: Instalar a partir de Arquivo -> Clique em procurar e selecione o arquivo CNVisual_0.1.tar.gz baixado.

Instruções de utilização:
  • Para carregá-lo digite no console:library(CNVisual)
  • Para ter acesso às informações sobre o pacote e suas funções digite: help(package = CNVisual)
  • Para iniciar o programa digite: CNVisual::JANELAPRINCIPAL()
